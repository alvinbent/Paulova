# Paunova Digital Clinic System

Repositorio central para la plataforma digital de **Dra Carolina Aguirre - Paunova Skin & Age Clinic**.

Este proyecto se gestionara por fases. En esta primera preparacion no se crea aun la pagina web; se deja listo el repositorio central, los agentes de apoyo, el plan maestro y la base de coordinacion para construir con calidad.

## Reglas de marca

- Nombre de la doctora: **Dra Carolina Aguirre**
- Nombre oficial del negocio: **Dra Carolina Aguirre - Paunova Skin & Age Clinic**
- No cambiar identidad visual, tono ni promesas medicas.
- No usar diagnostico automatizado ni prometer resultados garantizados.
- Separar datos clinicos, datos comerciales y datos publicitarios.

## Stack objetivo

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- GitHub como repositorio central
- Vercel como produccion
- Google Sheets como base de datos inicial
- Arquitectura preparada para Supabase/PostgreSQL
- WhatsApp Business Cloud API
- OpenAI API para asistencia de contenido, chatbot y cerebro publicitario
